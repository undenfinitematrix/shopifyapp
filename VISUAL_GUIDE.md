## Customization Examples

### Change Button Color

Find this line in `widget.js.jsx`:
```javascript
background-color: #25D366 !important;
```

Change to your preferred color:
```javascript
// Green variations
#25D366  (WhatsApp default)
#20ba5a  (Darker green)

// Or try other colors
#0099cc  (Blue)
#FF5722  (Deep Orange)
#7c3aed  (Purple)
#EC4899  (Pink)
```

### Change Button Position

```javascript
// Original (bottom-right)
bottom: 20px !important;
right: 20px !important;

// Bottom-left
bottom: 20px !important;
left: 20px !important;

// Top-right
top: 20px !important;
right: 20px !important;

// Adjust distance
bottom: 30px !important;  // Further from bottom
right: 30px !important;   // Further from right
```

## Common Questions

### Q: Where is the WhatsApp button?
**A:** Bottom-right corner of your store homepage. If not visible:
1. Check "Enable Widget" is toggled ON
2. Clear browser cache
3. Refresh page
4. Check phone number is saved with country code

### Q: How do I test it?
**A:** 
1. Click the green button
2. Should open WhatsApp with your message
3. If not working, check if WhatsApp is installed on your device

### Q: Can I move the button?
**A:** Yes! Edit `bottom` and `right` values in the CSS. See "Customization Examples" above.

### Q: Will it slow down my store?
**A:** No! The script is only 2KB and loads after page (using `defer`).

### Q: How many phone numbers can I use?
**A:** Currently, one per store. Multiple numbers coming in future updates.

### Q: Can I remove it anytime?
**A:** Yes! Just remove the script tag from theme.liquid or disable in app.
